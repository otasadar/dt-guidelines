Modals

Modals are part of Bootstrap. Documentation can be found here. Modal open event can be attached to a button or a link by adding data attributes: data-toggle="modal" data-target="#modal-id". Alternatively, modals initialized with the following script: $('.selector').modal();

Helper method that displays modal alerts is included into dt.core.js. Helper method is: /* type string required - default, success, info, warning, error message string required - alert message showIcon bool optional default_value = false - adds an success/info/etc. sign in front of the message showHeader bool optional default_value = false - determines if header need to be displayed title string optional default_value = '' - modal title displayed in header buttons mixed optional default_value = false - list of button objects: [{ text: 'Ok', classes: 'btn-submit', closeModal: false, click: false }, { text: 'Close', classes: 'btn-cancel', closeModal: true, click: false }, { text: 'My Button', classes: 'btn-default', closeModal: false, click: function() { alert('My Button clicked!'); } }] callback mixed optional default_value = false - false or function to be executed after modal is shown */ dt.core.modal.showAlert(type, message, showIcon, showHeader, title, buttons, callback);

Modal alerts: default, info, success, warning, error

Modal alerts with icon: default doesn't have icon!, info, success, warning, error

Modal alert with header: show modal

Modal alert with header and buttons: show modal

Modal alert with callback: show modal

Form

Form validation errors will be attached to the fields. To show errors in the form of tooltips just add "jquery-validate-tooltip" class to the form.

Date Picker

Date Picker utilizes jQuery UI datepicker component (component link). Date Picker initialized with the following script: $('selector').datepicker();

Autocomplete

Autocomplete utilizes jQuery UI autocomplete component (component link). Autocomplete initialized with the following script: var js_list = [ "Australia", "Austria", "Brazil", "Canada", "Chile", "China", "Denmark", "Egypt", "France", "Germany", "Greece", "India", ...]; $('selector').autocomplete({source: js_list});

Enhanced Select

Enhanced Select utilizes jQuery plugin "Chosen" (plugin link). Enhanced Select initialized with the following script: $('.selector').chosen();

 

Masked Input

Masked Input utilizes jQuery plugin "Masked Input" (plugin link). Masked Input initialized with the following script: $('.selector').mask("string mask i.e. 99/99/9999");
